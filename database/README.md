# Database Codes Location

The database codes (Migrations, Seeders, Factories) for this project are developed using Laravel.

The definitive and actively used versions of the database files are located in the `server/database/` directory. 

Copies of the migration and seeder files have been mirrored into this folder (`database/`) for your convenience. Please examine these mirrored files to verify the database structure. Remember that when you run `php artisan migrate`, it executes the files from the `server/database/migrations/` directory.

Models are located in `server/app/Models`.
